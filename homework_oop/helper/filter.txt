filter - оставляет только репозитории, удовлетворяющие условию
    1) filter <название_поля_целочисленного_типа> оператор_сравнения число
        Пример: filter size <= 100
    2.1) filter <название_поля_строкового_типа> alphabet оператор_сравнения строка
        Пример: filter language alphabet == TypeScript
    2.2) filter <название_поля_строкового_типа> length оператор_сравнения число
        Пример: filter name length < 20
    2.3) filter <название_поля_строкового_типа> contains строка
        Пример: filter description contains developer
    3) filter <название_поля_логического_типа> true / false
        Пример: filter has_wiki true
    4) filter <название_поля_типа_дата_и_время> оператор_сравнения дата_и_время
        Пример: filter created_at >= 2013-10-11T06:50:37Z
    5) filter topics contains строка
        Пример: filter topics contains education
    Можно написать несколько фильтров, тогда они применятся все одновременно.
    Важно: ни один фильтр не выполнится до сохранения файла!
